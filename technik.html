Technik

Neben den Basics mit HTML und CSS, gibt es zahlreiche Gadgets (Spielereien), mit denen die Webseite ansprechender gestaltet werden kann. Diese Gadgets werden mit CSS3 und manchmal auch mit Javascript umgesetzt. Hier werden drei häufig verwendete Effekte vorgestellt:

  1. Ein Unterstreichungs-Hover-Effekt für die Navigation
  2. Ein Scrollup-Button, der ab einer bestimmten Scrollhöhe angezeigt wird und per Klick zum Anfang der Seite scrollt
  3. Ein Suchfeld, dass sich vergrößert, wenn hineingeklickt wird

Navigation-Hover

Ein Hover-Effekt ist eine Formatierung, die angewandt wird, wenn die Maus sich über dem Element befindet. Der gehoverte Zustand wird im Stylesheet durch den Selektor :hover angesporchen.

Der Navigation-Hover-Stroke ist ein Effekt, bei dem der Link beim Hovern von der Mitte her nach außen unterstrichen wird.

Zunächst wird die Position der Links im Navigationsbereich auf relative gesetzt, damit sie für enthaltene Elemente als Elternelement fungieren können. Anschließend wird in den Links vor dem Text mit :before und :content neuer Inhalt erzeugt. Der Inhaltebereich ist leer, 100% breit und 1 Pixel hoch. Da der Link als Elternelement fungiert, hat der Inhaltsbereich die Breite des Links. Mit der Positionierungsart absolut und der Position bottom: 0

wird der neue Bereich am unteren Rand des Elternelementes ausgerichtet. Unter den Links erscheint eine Linie mit der angegebenen Hintergrundfarbe.

Die Linie wird mit der Anweisung scaleX(0) zunächst auf 0% Breite gesetzt. Beim Hovern wird sie durch scaleX(1) auf 100% ausgedehnt. Damit sich die Linie allmählich ausdehnt, wird eine Übergangszeit von 0.3 Sekunden angegeben.

HTML

<nav class="stroke">
<ul>
<li><a href="#">Konzeption</a></li>
...
</ul>
</nav>

CSS

nav.stroke ul li a,
nav.fill ul li a {
position: relative;
}
nav.stroke a:before {
content: "";
position: absolute;
width: 100%;
height: 1px;
bottom: 0;
background-color: rgba(40, 31, 55, 0.5);
visibility: hidden;
-webkit-transform: scaleX(0);
transform: scaleX(0);
-webkit-transition: all 0.3s ease-in-out 0s;
transition: all 0.3s ease-in-out 0s;
}
nav.stroke a:hover:before {
visibility: visible;
-webkit-transform: scaleX(1);
transform: scaleX(1);
}

Scrollup Button

Ein Scrollup Botton ist eine Schaltfäche, die am Seitenrand erscheint, sobald die Webseite nach unten gescrollt wurde. Ein Klick auf den Button scrollt die Seite an den Anfang.

Ähnlich wie beim Navigation-Hover-Stroke wird in das mit der Klasse scrollUp gekennzeichnete Element mit :before ein Button-Element eingefügt. Der Button ist zunächst ausgeblendet (display: none) und wird eingeblendet (display: block), sobald eine Scrollhöhe von 300 Pixeln erreicht wurde. Durch die Positionierungsart fixed bleibt der Button auch beim Scrollen in derselben vertikalen Position.

Das Einfügen des Buttons und das Scrollen bei Button-Klick werden durch das Javascript gadgets.js implementiert, das im head der Seite eingebunden werden muss.

HTML

<head>
<meta charset="UTF-8">
<title>HTML/CSS Workshop - </title>
...
<script type="text/javascript" src="js/gadgets.js"></script>
<link rel="stylesheet" type="text/css" href="style.css">
</head>
...
<footer class="scrollUp">
Impressum | Datenschutz
</footer>

CSS

#scrollUpBtn {
display: none;
position: fixed;
height: 50px;
width: 50px;
bottom: 130px;
right: 0px;
border: 1px solid rgb(252,214,229);
cursor: pointer;
padding: 15px;
background-color: rgba(252,214,229,0.5);
font-size: 24px;
}

Expandable Searchbox

Die Expandable-Searchbox ist ein Suchefeld, dass sich Vergrößert, wenn der Benutzer hineinklickt.

HTML

<nav id="nav-top">
<ul>
<li><a href="index.html">Home</a></li>
<li><a href="kontakt.html">Kontakt</a></li>
<li>
<form>
<input type="search" placeholder="Suche">
</form>
</li>
</ul>
</nav>

CSS

input {
 outline: none;
}
input[type=search] {
 -webkit-appearance: textfield;
 -webkit-box-sizing: content-box;
 font-family: inherit;
 font-size: 100%;
}
input::-webkit-search-decoration,
input::-webkit-search-cancel-button {
 display: none;
}
input[type=search] {
 background-image: url(../images/search-icon.png); /*Lupe*/
 background-repeat: no-repeat;
 background-size: 18px;
 background-position: 9px center;
 border: solid 1px rgba(255, 255, 255, 0);
 padding-left: 32px;
 width: 100px;
 font-size: 1rem;
 font-family: "Roboto", sans-serif;
 background-color: rgba(255, 255, 255, 0);
 -webkit-transition: all .5s;
 -moz-transition: all .5s;
 transition: all .5s;
}
input[type=search]:focus {
 width: 300px;
 padding: 6px 10px 6px 32px;
 border-color: #ddd;
}
input:-moz-placeholder {
  color: rgba(40, 31, 55, 0.9);
}
input::-webkit-input-placeholder {
  color: rgba(40, 31, 55, 0.9);
}

Hinweis

Für den Scrollup-Button wird das Javascript gadgets.js benötigt.

Alle Effekte sind in diese Webseite eingebaut, so dass sie auch gleich ausprobiert werden können.
